#  House Price Prediction

##  Problem Statement
The objective of this project is to analyze factors affecting house prices and build a machine learning model to predict house prices based on various features.

---

##  Dataset
The dataset includes features such as:
- Crime rate
- Number of rooms
- Distance from facilities
- Environmental factors (air quality, parks, waterbody)
- House price (target variable)

---

##  Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

##  Project Workflow

### 1. Data Cleaning
- Handled missing values
- Converted categorical variables into numerical format

### 2. Exploratory Data Analysis (EDA)
- Analyzed distributions using histograms
- Studied relationships using scatter plots
- Used correlation heatmap to identify important features

### 3. Key Insights
- House prices decrease as crime rate increases  
- Houses with more rooms tend to have higher prices  
- Environmental factors influence house prices  
- Distance-related features impact pricing  
- Availability of amenities increases property value  

---

##  Model Building
- Used **Linear Regression**
- Split data into training and testing sets

---

##  Model Performance
- **R² Score:** 0.64  
- **Mean Squared Error:** 26.03  

The model shows moderate performance and is able to capture key patterns in the data.

---

##  Prediction
- Predicted house prices on test data
- Compared actual vs predicted values using visualization

---

##  Conclusion
The project demonstrates an end-to-end data science workflow including data cleaning, analysis, visualization, and predictive modeling. Key factors such as number of rooms, crime rate, and environmental features significantly influence house prices.
